# Coin Flipper

A simple, interactive coin flipper website with animations and statistics tracking.

## Features

- Animated 3D coin flip
- Tracks heads/tails statistics
- Reset statistics option
- Keyboard support (press Spacebar to flip)
- Responsive design

## How to Use

1. Click the "Flip Coin" button or press Spacebar
2. Watch the coin flip animation
3. See the result (Heads or Tails)
4. View your flip statistics below

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and animations
- `script.js` - Coin flip logic and statistics

## Hosting on GitHub Pages

1. Push this repository to GitHub
2. Go to Settings > Pages
3. Select "main" branch as source
4. Your site will be live at `https://yourusername.github.io/coin-flipper/`
